Compétences Techniques et Expertise


En tant qu'un développeur junior

React

Librairie JavaScript côté client pour construire des interfaces utilisateur basées sur des composants et des applications monopages (SPA).

Laravel

Un framework web qui aide à créer des sites web et des applications web extensibles basés sur PHP à grande échelle.

MySQL

MySQL est un système de gestion de base de données relationnelle basé sur le langage SQL - Structured Query Language.

php

Hypertext Preprocessor, est un langage de script open source largement utilisé pour l'exécution côté serveur. Gratuit à télécharger et à utiliser, PHP facilite la génération de contenu dynamique, en en faisant un choix populaire pour le développement web.

Node Js

Un environnement d'exécution JavaScript côté serveur, construit sur le moteur JavaScript V8 de Google Chrome. Il permet d'exécuter du code JavaScript côté serveur.

MongoDB

MongoDB est un système de gestion de base de données NoSQL qui stocke les données au format BSON (Binary JSON) et permet l'exécution de requêtes.

MERISE

Méthodologie de conception de bases de données relationnelles. Elle propose une approche structurée pour analyser er modéliser des systèmes d'information.
